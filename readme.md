# 家庭药品有效期记录APP

这是一个操作极简的家庭药品有效期记录APP，专为中老年人设计，界面简洁明了，功能实用。

## 核心功能

- **添加药品信息**：输入药品名称、有效期（选择日期）、存放位置、数量，支持修改和删除
- **过期提醒**：药品到期前7天弹出提醒，已过期的药品在列表里标红
- **数据存储**：仅存在手机本地，不用登录，删除的药品可在"回收站"找回（保留7天）
- **亲人共享**：可以生成分享链接，方便子女提醒老人清理过期药物

## 界面特点

- **黄色底色**：温馨舒适，适合中老年人视觉
- **透明提醒框**：清晰显示过期提醒
- **大字体**：易于阅读
- **颜色对比明显**：过期药品标红，快过期药品正常显示

## 安装和使用步骤

### 方法一：在电脑上使用

1. **下载文件**：将所有文件（index.html、styles.css、script.js）下载到电脑上的同一个文件夹中
2. **打开应用**：双击 index.html 文件，在浏览器中打开即可使用

### 方法二：在手机上使用（推荐）

#### Android手机

1. **下载文件**：将所有文件下载到手机上的同一个文件夹中
2. **使用浏览器打开**：
   - 打开手机自带的浏览器（如Chrome、华为浏览器等）
   - 点击浏览器的"文件"或"本地文件"选项
   - 找到下载的文件夹，点击 index.html 文件
3. **添加到主屏幕**：
   - 打开应用后，点击浏览器右上角的三个点
   - 选择"添加到主屏幕"或"保存到主屏幕"
   - 点击"添加"，应用就会出现在手机主屏幕上，像普通APP一样使用

#### iPhone手机

1. **下载文件**：将所有文件下载到手机上的同一个文件夹中
2. **使用Safari浏览器打开**：
   - 打开Safari浏览器
   - 点击浏览器底部的"分享"按钮
   - 选择"文件"，将文件保存到"我的iPhone"或"iCloud Drive"
   - 在"文件"应用中找到保存的文件，点击 index.html 文件
3. **添加到主屏幕**：
   - 打开应用后，点击Safari浏览器底部的"分享"按钮
   - 向上滑动，找到"添加到主屏幕"
   - 点击"添加"，应用就会出现在手机主屏幕上，像普通APP一样使用

## 使用指南

### 添加药品

1. 点击首页右下角的"+ 添加药品"按钮
2. 输入药品名称（例如：布洛芬）
3. 选择有效期（点击日期选择器，不用手动输入）
4. 输入存放位置（例如：客厅抽屉）
5. 输入数量
6. 点击"保存"按钮

### 编辑或删除药品

1. 在首页点击要编辑的药品
2. 在编辑页面修改药品信息，点击"更新"按钮保存
3. 或点击"删除"按钮删除药品（删除后可在回收站找回）

### 查看过期提醒

- 首页上方的透明提示框会显示过期和快过期的药品
- 已过期的药品会标红显示
- 药品到期前7天会弹出提醒

### 使用回收站

1. 点击首页右上角的"设置"按钮
2. 在设置页面下方的"回收站"中查看删除的药品
3. 点击"恢复"按钮可以找回删除的药品
4. 点击"清空回收站"按钮可以清空回收站（删除的药品保留7天）

### 与亲人共享

1. 点击首页右上角的"设置"按钮
2. 在设置页面下方的"亲人共享"部分，点击"生成分享链接"
3. 复制生成的链接，发送给亲人
4. 亲人打开链接后，可以查看药品信息并选择导入到自己的应用中

## 注意事项

- 数据仅存储在手机本地，不会上传到服务器
- 删除的药品在回收站中保留7天，7天后会自动清除
- 为了确保提醒功能正常工作，建议定期打开应用
- 本应用完全免费，无广告，无付费功能

## 技术说明

- 本应用使用纯前端技术开发，包括HTML、CSS和JavaScript
- 数据存储使用浏览器的localStorage功能
- 界面设计响应式，适配不同屏幕尺寸
- 支持离线使用，无需联网

希望这个应用能帮助您和家人更好地管理家庭药品，避免使用过期药物！
